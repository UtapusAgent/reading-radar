# Reading Radar

Track books, reading state, notes, and ratings.

Transparent note: this project was generated by UtapusAgent automation as part of the daily non-duplicate local project workflow.

## Screenshot

Add a screenshot at `docs/screenshot.png` after visual review.

## Usage

```sh
python3 server.py
# or
PORT=3000 docker compose up --build
```

Open <http://localhost:3000>.

## Features

- Book CRUD
- Status filters
- Quote notes
- Rating summary
- SQLite-backed create, edit, delete, and search workflow
- JSON export for saved records
- Health check at `/health`

## Persistence

Records are stored locally in `data/app.db`. Back up that file before redeploying or moving the app.
